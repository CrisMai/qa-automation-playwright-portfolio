# QA Automation Portfolio: Playwright + TypeScript

This repository is a technical demonstration of an End-to-End (E2E) test automation framework built with **Playwright** and **TypeScript**.

## 🚀 Objective
To demonstrate the application of best practices in test automation, including project structure, functional testing, and Continuous Integration (CI).

## 🛠 Tech Stack
- **Language:** TypeScript
- **Framework:** Playwright
- **CI/CD:** GitHub Actions (Automated workflow configured to run tests on every push)
- **Pattern:** Page Object Model (POM)

## 📋 How to Run
1. Clone the repository:
   `git clone https://github.com/CrisMai/qa-automation-playwright-portfolio.git`
2. Install the dependencies:
   `npm install`
3. Install the browsers:
   `npx playwright install`
4. Execute the tests:
   `npx playwright test`

## 📊 Project Status
This project utilizes **GitHub Actions** to ensure code quality and test reliability on every commit.
